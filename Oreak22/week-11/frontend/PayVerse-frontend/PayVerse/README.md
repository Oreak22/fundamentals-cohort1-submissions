# PayVerse Frontend (React + Vite + TypeScript)

This frontend consumes the PayVerse backend using REST APIs, Axios, and TailwindCSS for styling.

## Tech Stack
- React + Vite
- TypeScript (TSX)
- TailwindCSS
- Axios
- React Router

## Pages
1. **Transactions Page**
   - Displays list of transactions.
   - Loading + error state.

2. **Health Check Page**
   - Displays backend API status.

## Setup Instructions

### 1. Install Dependencies
npm install

### 2. Create .env
VITE_API_URL=http://localhost:4000/api

### 3. Run Dev Server
npm run dev

### 4. Build for Production
npm run build
